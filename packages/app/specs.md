Escenario 1: (menu_screen.py)
  El usuario debe poder abrir la aplicación.
  El usuario debe poder abrir explorador archivos, seleccionar rom y cargarlo en la app.
  El bótón de elegir servidor debe desbloquearse y debe aparecer mensaje indicando nombre del archivo cargado.
  El botón del usuario debe cargar una lista de los servidores disponibles.
  El usuario debe poder elegir servidor en la lista, cerrando la ventana emergente e indicando en un mensaje el servidor elegido.
  El botón de jugar debe desbloquearse y el usuario debe pasar al escenario 2.

Escenario 2: (emulator_screen.py)
  Al entrar al escenario 2, automaticamente hay una emulación en curso del rom seleccionado.
  Hay salida de video.
  Hay salida de audio.
  El gamepad responde.
  El usuario debe poder abrir el chat.
  El usuario debe poder escribir en el chat y ver reflejados sus mensajes.
  El usuario debe poder cerrar el chat.
  El usuario debe poder abrir el menu de opciones.
  El usuario debe poder seleccionar la opción cable link y abrir la interfaz link.
  El usuario debe poder escribir y borrar.
  El usuario debe poder conectar , debe cerrarse la interfaz y aparecer el mensaje de waiting, con tx 0.
  El usuario debe poder conectar con otra id.
el usuario debe poder volver a abrir el chat y ver el mensaje de antes y escribir otro.
